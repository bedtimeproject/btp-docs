Procedures

Red Folder

This is the procedure for creating and approving content to put on the site. Following this convention will reduce downtime and increase quality of the work we put out.

Red Folder Steps:

  1. Create Content
  2. Create task in Trello's Red Folder List
  3. Another Admin will review, and provide notes if necessary
  4. When all looks good, add approved label to task
  5. Once approved, the asset is good for use

Adding Colors/Fonts to Theme (style guide)

Follow the same process as the Red Folder, described above.

Provide an explanation explaining why the color or font you chose should be added to our brand colors/fonts
